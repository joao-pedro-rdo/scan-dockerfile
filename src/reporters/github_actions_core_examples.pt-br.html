🛠️ Métodos Utilitários do Buffer:

core.summary.clear()
core.summary.clear()

Esvazia o buffer do summary E apaga o arquivo de summary no disco

core.summary.stringify()
const summaryContent = core.summary.stringify()

Retorna o conteúdo atual do buffer como uma string

core.summary.isEmptyBuffer()
const isEmpty = core.summary.isEmptyBuffer()

Verifica se o buffer do summary está vazio

core.summary.emptyBuffer()
core.summary.emptyBuffer()

Reseta o buffer sem escrever no arquivo de summary no disco

core.summary.write()
core.summary.write({overwrite: true})

Escreve o texto do buffer no arquivo e esvazia o buffer. Opcionalmente sobrescreve todo o conteúdo existente

💡 Como usar no GitHub Actions:

import * as core from '@actions/core'

// Exemplo prático com cores nos logs
core.info('\u001b[32m✅ Build completed successfully!')
core.info('\u001b[33m⚠️  Warning: Some dependencies are outdated')
core.info('\u001b[31m❌ Tests failed in module: authentication')

// Usando cores para destacar informações importantes
core.info('\u001b[1;36mDeployment Status Report\u001b[0m')
core.info('\u001b[32m• Frontend: \u001b[1mDeployed\u001b[0m')
core.info('\u001b[32m• Backend: \u001b[1mDeployed\u001b[0m') 
core.info('\u001b[31m• Database: \u001b[1mFailed\u001b[0m')

// Tabela de resultados de testes com cores
const testResults = [
  [{data: 'Test Suite', header: true}, {data: 'Status', header: true}, {data: 'Duration', header: true}],
  [{data: 'Unit Tests'}, {data: '✅ Pass'}, {data: '2.3s'}],
  [{data: 'Integration Tests'}, {data: '✅ Pass'}, {data: '5.1s'}],
  [{data: 'E2E Tests'}, {data: '❌ Fail'}, {data: '1.2s'}]
]

core.summary
  .addHeading('🚀 Relatório de Deployment', '1')
  .addSeparator()
  .addRaw('Status: ', false)
  .addRaw('✅ Sucesso\n', true)
  .addBreak()
  .addQuote('Deploy realizado com sucesso!', 'Sistema')
  .addCodeBlock('npm run build && npm run deploy', 'bash')
  .addHeading('📊 Resultados dos Testes', '2')
  .addTable(testResults)
  .write() // Salva o summary no GitHub

// Verificar se o buffer está vazio
if (!core.summary.isEmptyBuffer()) {
  core.info('\u001b[32m✅ Summary criado com sucesso!')
}
