üõ†Ô∏è Buffer Utility Methods:

core.summary.clear()
core.summary.clear()

Empties the summary buffer AND wipes the summary file on disk

core.summary.stringify()
const summaryContent = core.summary.stringify()

Returns the current summary buffer as a string

core.summary.isEmptyBuffer()
const isEmpty = core.summary.isEmptyBuffer()

Checks if the summary buffer is empty

core.summary.emptyBuffer()
core.summary.emptyBuffer()

Resets the summary buffer without writing to the summary file on disk

core.summary.write()
core.summary.write({overwrite: true})

Writes text in the buffer to the summary buffer file and empties the buffer. Optionally overwrites all existing content
